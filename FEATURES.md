### Goal: Increase Signups

- [x] "Powered by Trustwall" click-through landing page — converts widget viewers into signups
- [x] Testimonial request message generator — pre-written email/WhatsApp/LinkedIn templates with link
- [x] QR code generator — scannable QR code for testimonial collection link
- [x] Social media testimonial image generator — OG image API + share button on approved testimonials

- [x] Analytics per space — view counts, submission rates, approval rates

- [x] Testimonial tags — organize testimonials by category/product
- [x] Custom thank-you page — personalize the post-submission experience (settings page + redirect URL)
- [x] Auto-approve rules — set min rating threshold to auto-approve (in Space Settings)

### Goal: Differentiate from Competitors
- [x] AI testimonial summary — extractive highlight generator with "Generate highlight" button
- [x] Star rating summary widget — show aggregate rating (4.8/5 from 24 reviews)
- [x] Rich testimonial form — optional company logo URL and social profile link fields
- [x] Multi-language support — auto-detect language on submission, language badge on cards
- [x] Testimonial video support — paste YouTube/Loom/Vimeo URLs, embedded player in dashboard & Wall of Love
